# Streamify Android

Open this `android` directory in Android Studio.

Before running the app, update `serverUrl` in:

`app/src/main/java/com/streamify/MainActivity.kt`

Use the local IP address of the machine running the Streamify server, for example:

`http://192.168.1.23:3000`

The web/server app must be running with `npm run dev` or `npm start`, and the Android device must be on the same Wi-Fi network.
